Суть проекта: парсинг сайта Habr для считывания тегов статьей, создание базы данных тегов, добавление тегов за определеный временной интервал в базу данных, вывод данных из базы, вывод данных в csv файл, представление частоты использования тегов в виде гистограммы.
Парсинг одних и тех же статей не заносит повторно данные в базу.


Сценарий запуска проекта:

python MyProject.py createbd --pages=5
-> База успешно создана

python MyProject.py inform
-> Сейчас в базе имеются данные с 7 января 2020 по 10 января 2020

python MyProject.py parsstor --fromday=4 --frommonth=января --fromyear=2020 --today=7 --tomonth=января --toyear=2020 
->В базу добавлены данные по тегам c 4 января 2020 года по 7 января 2020 года

python MyProject.py inform
->Сейчас в базе имеются данные с 4 января 2020 по 10 января 2020

python MyProject.py getdata
->*Вывод имеющихся данных*

python MyProject.py csv --fromday=5 --frommonth=января --fromyear=2020 --today=7 --tomonth=января --toyear=2020
->csv файл создан

python MyProject.py plot --fromday=6 --frommonth=января --fromyear=2020 --today=7 --tomonth=января --toyear=2020 5
->*Вывод гистограммы для 5 самых популярных тегов с 6 января 2020 года по 7 января 2020 года*
