Суть проекта: парсинг сайта Habr для считывания тегов статьей, создание базы данных тегов, добавление тегов за определеный временной интервал в базу данных, вывод данных из базы, вывод данных в csv файл, представление частоты использования тегов в виде гистограммы.
Парсинг одних и тех же статей не заносит повторно данные в базу.


Сценарий запуска проекта:

python MyProject.py createbd --pages=5
-> База тегов успешно создана по 5 последним страницам

python MyProject.py inform
-> Сейчас в базе имеются данные с 16 января 2020 по 17 января 2020

python MyProject.py parsstor --fromdate=2020-01-13 --todate=2020-01-15
->В базу добавлены данные по тегам c 13 января 2020 года по 15 января 2020 года

python MyProject.py inform
->Сейчас в базе имеются данные с 13 января 2020 по 17 января 2020

python MyProject.py getdata
->*Вывод имеющихся данных*

python MyProject.py csv --fromdate=2020-01-14 --todate=2020-01-16
->csv файл создан

python MyProject.py plot --fromdate=2020-01-14 --todate=2020-01-16 5
->*Вывод гистограммы для 5 самых популярных тегов с 14 января 2020 года по 16 января 2020 года*
